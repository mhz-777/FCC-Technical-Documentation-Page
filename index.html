HTML: HyperText Markup Language

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

HTML uses "markup" to annotate text, images, and other content for display in a Web browser. HTML markup includes special "elements" such as <head>, <title>, <body>, <aside>, <nav>, <header>, <footer>, <article>, <section>, <p>, and many others.

An HTML element is set off from other text in a document by "tags", which consist of the element name surrounded by "<" and ">". The name of an element inside a tag is case insensitive. That is, it can be written in uppercase, lowercase, or a mixture. For example, the <title> tag can be written as <Title>, <TITLE> or in any other way. However, the convention and recommended practice is to write tags in lowercase.

Get Started

Now that you are up to speed with the basics of HTML, let's take a look at some basic code. Below is an example of HTML5 "boiler plate code". This is the foundation of all HTML5 webpages:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta http-equiv="X-UA-Compatible" content="ie=edge"> <title><title> <link rel="stylesheet" href="style.css"> </head> <body> </body> </html>

After creating an index.html file in vscode, go to the first line and type the following: html:5. You should see the above boiler plate code now in your own workspace thanks to Visual Studio Codes built-in Emmet extension. Go ahead and try give your website a document title. If you get stuck, try placing it within the <title> tag.

If you navigate to the directory where you created the project and click the index.html file, you should see your default browser open up to the webpage. Making changes within your IDE and hitting refresh in the browser will reveal these changes. Try placing a header on your webpage with the title "Hello World!". If you get stuck, refer below to the code snippet:

<body> <h1>Hello World!</h1> </body>

You now know the basic building blocks of an HTML webpage! Experiment with the various other tags and see if you can make something else. Should you get stuck, or not know how to do something refer to some of the guides below.
